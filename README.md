# EstrellaNova
Es un juego indie para PC y móvil en el que se debe ir superando los distintos niveles. Cada nivel cuenta con una complejidad mayor (al estar las estrellas cada vez mas lejos).

La monetización de este juego se basa en la venta de objetos excluisvos, como skins o efectios especiales que mejoran la experiencia visual del usuario.

Para este juego se ha elegido la licencia GNU General Public License v3.0 para las mecánicas básicas del juego, mientras que los objetos exclusivos con los que monetizaremos el juego serán recursos propietarios.

He elegido esta licencia porque permite que otros desarrolladores mejoren el código sin que tengan que liberar todo el proyecto. De esta forma mi juego cuenta con trnsparencia en el código, pero protege mis assets exclusivos.

Con esta licencia, doy derechos como el de usar mi proyecto incluso para fines comerciales, modificar y distribuir mejoras.

Las limitaciones son que no me hago responsable de garantizar que el softaware funcionará correctamente en todos los casos y por tanto no soy responsable de los daños que se puedan causar al usar el código.

Económicamente puedo monetizar mi juego con objetos y aspectos exclusivos mientras que permito que todos puedan disfrutar de mi juego de forma gratuita y compartiendo la parte técnica para la comunidad de desarrolladores, por lo cual también es ético.
