# EstrellaNova
Es un juego indie para PC y móvil en el que se debe ir superando los distintos niveles. Cada nivel cuenta con un nivel de complejidad mayor (al estar las estrellas cada vez mas lejos).

La monetización de este juego se basa en la venta de objetos excluisvos, como skins o efectios especiales que mejoran la experiencia visual del usuario.
